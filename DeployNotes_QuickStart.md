# DeployNotes - Quick Start Guide

## 📥 Get the App (Pick One Method)

### Method A: Direct Share
1. Receive **DeployNotes.html** via WhatsApp/email
2. Save to your phone's Files app
3. Open in browser → Done!

### Method B: Team Link
1. Get the URL from your team lead
2. Open in Safari (iPhone) or Chrome (Android)
3. Install to home screen (see below)

---

## 📱 Install to Home Screen

### iPhone
1. Open link in **Safari**
2. Tap **Share** button (box with arrow)
3. Tap **"Add to Home Screen"**
4. Tap **Add**

### Android
1. Open link in **Chrome**
2. Tap **⋮ menu** (3 dots)
3. Tap **"Add to Home Screen"**
4. Tap **Add**

---

## 🔐 First Time Setup

1. **Create PIN** - Enter 4 digits, then confirm
2. **Open Settings** - Tap "⚙️ Settings & Cloud Setup"
3. **Set Deployment** - Enter your mission name
4. **Save** - Tap "Save Settings"

---

## 📝 Daily Use

### Start of Day
- Open app → Enter PIN
- Set date, day number, location

### During the Day
- **🚗 Journey** - What you saw traveling
- **👥 People** - Who you met + consent level
- **💪 Impact** - Numbers & activities
- **📸 Media** - Capture photos/videos
- **📖 Stories** - Human stories
- **🏥 Health** - Risks observed

### End of Day
- **💭 Reflection** - Write your thoughts
- **☁️ Upload** or **📄 Export**
- **🔒 Lock App**

## ✅ Consent Levels

| Icon | Level | Use |
|------|-------|-----|
| ✅ | Full | Name, photo, quotes OK publicly |
| ⚠️ | Partial | Anonymous only |
| 🔒 | Internal | Team use only |

## 💾 Save Your Work

| Button | When to Use |
|--------|-------------|
| ☁️ Upload | Have wifi → Send to cloud |
| 📄 Export | No wifi → Save file locally |
| 🗑️ Clear | Ready for new day |

---

## 🚨 Important Notes

- **PIN** - Write it down! No recovery option
- **Auto-save** - Data saves as you type
- **Offline** - Works without internet
- **Consent** - Always get permission before photos

---

## ❓ Problems?

| Issue | Solution |
|-------|----------|
| Won't install | iPhone=Safari, Android=Chrome |
| Camera won't work | Check permissions, try Gallery |
| Upload failed | Data saved locally, try later |
| Forgot PIN | Must clear data & restart |
